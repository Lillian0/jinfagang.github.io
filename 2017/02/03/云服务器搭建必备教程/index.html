然后配置nginx，哦对了，还有一步，虽然服务器的防火墙配置了，但是你的云管理中心的安全组也要配置，不知道阿里云有没有，腾讯云是有安全组的，你要在安全组中增加开放80端口的规则，或者直接用默认的开放所有端口。

    如果你ostgreslq移植无法变成utf-8，那么后面会很蛋疼，这样你修改系统的locale设置，一般来说是这样的：

sudo apt-get -y remove –purge postgresql-9.5
sudo apt install postgresql

sudo -u postgres createdb -O root oumenglite
```
