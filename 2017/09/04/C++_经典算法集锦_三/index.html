Algorithm 6. 字符串算法

既然字符串经常出现，那我就记录一些字符串算法。
首先是，用hash表查找，比如给一个字符串fierjggooi要找出第一个重复的字符，这里应该是i，因为i出现了两次，而且是首次出现。要怎么写呢？

其实也非常简单，只需要写一个hash表，记录每个字符对应出现的次数即可。说白了就是一个字典，只不过这里更加底层一些。
